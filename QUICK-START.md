# ⚡ Quick Start - Deploy trong 5 phút

## 🎯 Cách Nhanh Nhất: Vercel Drag & Drop

### Bước 1: Đăng ký Vercel
1. Truy cập: https://vercel.com/signup
2. Đăng nhập bằng GitHub/Google/Email

### Bước 2: Deploy
1. Vào: https://vercel.com/new
2. Kéo thả toàn bộ thư mục `nam` vào
3. Đợi 30 giây
4. Nhận link website! 🎉

**Link sẽ có dạng**: `https://lux-lady-jewelry.vercel.app`

---

## 🔄 Cách Tự Động: Deploy qua GitHub

### Bước 1: Tạo Git Repository
```bash
git init
git add .
git commit -m "Initial commit"
```

### Bước 2: Push lên GitHub
1. Tạo repo mới: https://github.com/new
2. Đặt tên: `lux-lady-jewelry`
3. Chạy lệnh (thay YOUR_USERNAME):
```bash
git remote add origin https://github.com/YOUR_USERNAME/lux-lady-jewelry.git
git branch -M main
git push -u origin main
```

### Bước 3: Connect với Vercel
1. Vào: https://vercel.com/new
2. Chọn "Import Git Repository"
3. Chọn repo `lux-lady-jewelry`
4. Click "Deploy"
5. Xong! 🚀

**Lợi ích**: Mỗi lần push code, Vercel tự động deploy lại!
